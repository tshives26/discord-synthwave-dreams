# Synthwave Dreams Theme

## GitHub Repository

**Repository URL:** https://github.com/tshives26/discord-synthwave-dreams

**Author:** SummitByte  
**Version:** 1.0.0

## Description

A retro-futuristic synthwave theme with vibrant neon colors, grid lines, and 80s aesthetic. Built on ClearVision framework.

## Installation

1. Download the `SynthwaveDreams.theme.css` file from the repository
2. Place it in your BetterDiscord themes folder
3. Enable it in BetterDiscord settings

## Auto-Update

This theme uses GitHub imports for automatic updates. When you reload Discord or the theme, it will pull the latest version from GitHub.

## Framework Files

- `main.css` - Main theme framework
- `betterdiscord.css` - BetterDiscord-specific styles

## Assets

- Background images and animated GIFs are stored in the `assets/` folder
